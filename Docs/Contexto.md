# HealthView

## Objetivo
- Um sistema que salva todo o histórico médico de um paciente

## Contexto
- A ferramenta tem como objetivo juntar dados médicos de forma que fique fácil seguir com tratamentos ao analisar todo os histórico médico de paciente
- Um problema muito comum é que exames, laudos e percepção de médicos se percam com o passar dos anos, oque dificulta a análise correta dos dados

## Requisitos já elicitados
- Cadastro de usuário
- Upload de arquivos manualmente pelo usuário
- Cadastro de usuários com informações pertinentes ao sistemas - Adição de documentos digitais pelas Clínicas e médicos através do portal 
- Adição do comentários de outros médicos com acesso as informações do paciente - Apresentação dos dados do usuário de forma simples e objetiva 
- Apresentação detalhada do histórico do paciente
- Filtro do histórica por área médica
